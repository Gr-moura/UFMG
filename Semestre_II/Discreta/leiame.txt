Sobre o ProgramaEste programa em linguagem C utiliza as bibliotecas padrão <stdio.h> e <stdlib.h> para realizar suas operações. Ele implementa funcionalidades com uso de recursão e, durante sua execução, gera três arquivos de saída: i.txt, ii.txt e iii.txt. Estes arquivos são apenas criados e preenchidos com as sequências dos fractais gerados.Requisitos do Ambiente	* Compilador: GCC (GNU Compiler Collection), versão 4.8 ou superior.  	* Sistema Operacional: O programa é multiplataforma e pode ser executado em sistemas Linux, Windows ou macOS.  	* Bibliotecas Necessárias: Apenas as bibliotecas padrão <stdio.h> e <stdlib.h> são requeridas.Como Compilar	1. Certifique-se de que o compilador GCC está instalado e configurado no seu sistema.		* gcc --version        	2. Compile o arquivo-fonte com o seguinte comando:		* gcc -o TP TP.cComo Executar	1. Após a compilação, execute o programa com:		* ./TP    	2. O programa aceitará entradas somente pela stdin (entrada padrão). Certifique-se de fornecer os dados corretamente, ou seja, digite apenas o estágio desejado para os fractais.    	3. Durante a execução, três arquivos serão gerados no diretório atual:		* i.txt 		* ii.txt 		* iii.txt	Esses arquivos conterão a sequência de caracteres de cada fractal, de acordo com o estágio requerido pela entrada.
